ASRs Consideradas En El Sprint 3

En esta sección presentamos los ASR consideradas para satisfacer los requerimientos de disponibilidad de la arquitectura. El objetivo es determinar los puntos de sensibilidad para determinar si las decisiones de arquitectura que se han tomado van por buen camino.
ASRs Consideradas

Código Descripción
ASR02 Como propietario, cuando cargue los datos de parqueaderos dado que el sistema opera en modo normal quiero que aparezca la oferta para que los clientes puedan visualizarlas disponibles. Esto debe suceder en el 99,99 % de las veces.​
ASR20 Como usuario, cuando aplique a una reserva dado que el sistema opera en modo sobrecarga quiero poder reservar un parqueadero para dejar mi carro cerca al trabajo. Esto debe suceder el 99.999% de las veces.
ASR21 Como usuario, cuando realice una búsqueda dado que el sistema opera normalmente quiero buscar parqueaderos dados los criterios: (Dirección, zona, tipo vehículo) para ver los parqueaderos más cercanos. Esto debe suceder el 99.999% de las veces.
